# Probabilistic Inference & Language Modeling System

AI and NLP system built with Python implementing Bayesian Network inference, probabilistic reasoning, N-gram language modeling, text generation, and perplexity evaluation.

This project focuses on probabilistic graphical models, uncertainty-aware reasoning, statistical language modeling, and foundational natural language processing techniques used in modern AI systems.

---

## Features

- Bayesian Network construction and probabilistic inference
- Conditional probability modeling and reasoning
- Hidden-state localization under noisy observations
- Unigram, bigram, and trigram language models
- Statistical text generation pipelines
- Perplexity evaluation for language modeling performance
- Probabilistic agent reasoning in partially observable environments

---

## Technical Highlights

### Bayesian Network Inference
Implemented probabilistic graphical models using:
- conditional probability tables
- probabilistic dependencies
- Bayesian inference
- uncertainty-aware reasoning

to model and evaluate complex event relationships.

### Probabilistic Agent Reasoning
Developed inference-based agent systems capable of:
- reasoning under uncertainty
- interpreting noisy observations
- estimating hidden states
- updating probability distributions dynamically

within partially observable environments.

### Statistical Language Modeling
Built N-gram language models including:
- unigram models
- bigram models
- trigram models

to generate text sequences and evaluate prediction quality using perplexity metrics.

### NLP & Text Generation
Implemented statistical NLP pipelines for:
- token probability estimation
- sequence prediction
- probabilistic text generation
- language uncertainty evaluation

using frequency-based modeling techniques.
